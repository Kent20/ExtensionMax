# MAX Chat Indexer

Расширение для Chrome: локально индексирует историю сообщений в веб-версии MAX (web.max.ru) и даёт полнотекстовый поиск по всем чатам — поверх встроенного поиска MAX, который ищет плохо.

## Как это работает

- Полностью автоматически: открыл чат → расширение в фоне прокручивает историю вверх и сохраняет все сообщения в IndexedDB (локально, никуда не уходит). Каждый чат бэкфилится один раз, дальше новые сообщения подхватываются live через MutationObserver.
- В правом нижнем углу страницы — маленькая иконка-лупа. Пока чат индексируется, она мягко пульсирует амбером.
- Клик по иконке (или **Ctrl+K** / **⌘+K** с любого места страницы) открывает оверлей поиска (command palette в стиле Spotlight) прямо на странице: вводишь текст — сразу видишь аватар, чат, время и подсвеченные совпадения. ↑/↓ — навигация, Enter/клик — перейти в чат, Esc — закрыть.
- В футере оверлея — **«Просканировать все чаты»**: по очереди открывает все личные чаты из списка слева (группы и каналы пропускает — отличает их по скрытой accessibility-метке «канал, »/«группа, » перед именем в списке) и индексирует каждый. ⚠️ Открытие чата помечает непрочитанные сообщения как прочитанные у собеседника — это поведение самого MAX, расширение его не отключает, поэтому перед запуском просит подтверждение. Остановить можно повторным кликом по иконке-лупе во время сканирования.

## Установка (распакованное расширение)

1. Открой `chrome://extensions`.
2. Включи «Режим разработчика» (переключатель справа сверху).
3. «Загрузить распакованное расширение» → выбери папку `ExtensionMax`.
4. Открой `https://web.max.ru`, зайди по очереди в чаты, которые хочешь искать — индексация начнётся сама.
5. Жми иконку-лупу в правом нижнем углу и ищи.

## Хранение данных

Сообщения хранятся в IndexedDB на стороне расширения (origin `chrome-extension://...`), не на сайте и не во внешних сервисах.

## Известные ограничения

- Селекторы завязаны на текущую (не хэшированную) разметку MAX: `.openedChat`, `.scrollListScrollable`, `.bubbleContent`, `.item`, `[data-bubbles-variant]`. Если MAX переименует эти классы при обновлении — индексация перестанет находить сообщения (заметно по тому, что иконка никогда не начинает пульсировать в открытом чате).
- В групповых чатах отправитель входящих сообщений определяется как название чата (т.е. имя группы), а не реальное имя автора — в сохранённой верстке группового чата с разными отправителями не было, чтобы понять точный селектор имени.

## Сброс / очистка

ПКМ по иконке расширения → «Параметры»: там можно полностью удалить собранный индекс.
